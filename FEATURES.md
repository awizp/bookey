  Bookey is a centralized platform to organize and manage your reading journey.

### About

Bookey is a web application designed to organize and manage books in one centralized platform. It provides readers with a
personalized space to track their reading journey, create custom collections, and maintaining a structured library.

  * Centralized book listing platform
  * Personalized reading collections
  * Simple and structured user experience

### The Problem Readers face scattered book tracking

1. Fragmented Platforms - Books exist across multiple platforms like Goodreads, Amazon, and personal spreadsheets
2. No Unified System - No single place to manage all reading lists and track progress efficiently
3. Lost Momentum - Difficult to track reading progress and remember what books you've read or want to read

### The Solution

* All Books in One Platform
    - Centralize your entire collection in a single, accessible location
* Organized Lists
    - Create custom categories like Read, Currently Reading, and Want to Read
* Easy Tracking
    - Monitor reading progress and update statuses with simple clicks

### Technology Stack

1. React.js - Frontend UI library creating dynamic, component based user interface with responsive design and state management
2. Node.js - Server runtime enabling fast, non-blocking I/O operations and handling concurrent connections efficiently
3. Express.js - Backend framework handling API routes, request processing, and business logic with efficient middleware architecture
4. MongoDB - Flexible NoSQL database for storing book data, user information, and reading progress in a scalable document structure

### Key Features

1. Centralized Book Library
    - Store all your books in one organized digital collection
2. Custom Reading Lists
    - Create personalized categories and tags for better organization
3. Complete Management
    - Add, update, and manage books with full control over your data
4. Responsive UI
    - Clean, intuitive interface that works seamlessly across all devices
5. Book and Genre Clubs
    - Create clubs with like minded people to log daily and streak

### How Bookey Works

1. Log In - User authenticates and accesses personalized dashboard
2. Add Books - Search and add books to your personal library
3. Categorize - Organize books into custom reading lists
4. Track Progress - Update reading status and track completion
5. Manage All - Access everything from unified dashboard
